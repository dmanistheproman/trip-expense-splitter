Trip expense tracker

Local test mode with browser storage only.

This version runs without the Node server or PostgreSQL. It stores data in your browser and is useful for quick testing.

How it works

  1. Add everyone on the trip.
  2. Record each expense and choose the split.
  3. All data is stored in this browser only.
